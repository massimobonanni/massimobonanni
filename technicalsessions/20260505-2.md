# One CLI to Deploy Them All: Building Smarter on Azure with azd
##  05/05/2026 - European Cloud Summit 2026 - Colonia
### Abstract 
Meet the command-line companion that’s redefining how developers build for the cloud. The Azure Developer CLI (azd) simplifies the entire application lifecycle — from provisioning infrastructure and deploying code to setting up CI/CD — all through a single, intuitive workflow.
In this session, you’ll discover how azd empowers developers to go from local code to running cloud apps in minutes. 

We’ll walk through real-world demos that show how to:

- Scaffold cloud-native projects instantly using ready-to-go templates.
- Provision infrastructure with Infrastructure as Code (IaC).
- Deploy and manage applications seamlessly across environments.
- Integrate effortlessly with GitHub Actions or Azure DevOps pipelines.

Whether you’re a solo developer or part of a large engineering team, you’ll learn how azd brings consistency, automation, and speed to every stage of your cloud journey.
Takeaway: One CLI. Every stage. Infinite productivity.
