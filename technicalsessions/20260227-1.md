# Zero setup, zero scuse: sviluppare ovunque con GitHub Codespaces
##  27/02/2026 - SharpCoding 2026 - Roma
### Abstract 
Laptop scarico? Tablet alla mano? Solo un browser a disposizione? Nessun problema. Con GitHub Codespaces il tuo ambiente di sviluppo vive nel cloud, pronto a partire in pochi secondi e identico per tutto il team. In questa sessione vedremo come costruire workspace riproducibili, sicuri e veloci, sfruttando l’integrazione con VS Code e i dev container. Chiuderemo con consigli pratici per migliorare performance, gestire risorse e tenere sotto controllo i costi. Una demo dinamica per chi vuole lavorare davvero “anywhere” — senza compromessi.
