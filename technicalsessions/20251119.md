# Coding anywhere: sviluppare ovunque con GitHub Codespaces
##  19/11/2025 - Sharp Coding Meetup - Roma
### Abstract 
Laptop, tablet o Chromebook — con GitHub Codespaces, il tuo ambiente di sviluppo è dove sei tu.
Questa sessione mostra come rendere il tuo workspace veramente portatile, sicuro e veloce, grazie all’integrazione tra Codespaces, VS Code.
Faremo insieme un tour completo: dal provisioning alla personalizzazione, fino ai trucchi per ottimizzare le performance e ridurre i costi.
Una demo “nomade”, ideale per chi vive tra coworking, aerei e browser tabs aperti ovunque.
