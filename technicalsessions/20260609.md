# GitHub Spark, il vibe coding secondo GitHub!
##  09/06/2026 - #4AI - Pescara
### Abstract 
Vibe Coding e' una delle buzzword del momento! Costruire un'applicazione solamente descrivendola....il sogno di tutti. Con GitHub Spark puoi descrivere ciò che vuoi costruire e ottenere in pochi minuti un’app funzionante, con interfaccia, logica e deploy inclusi. Durante la sessione costruiremo un’app dal vivo, la modificheremo “parlandoci sopra” e scopriremo come Spark cambia il modo di prototipare e sperimentare nuove idee — senza rinunciare al controllo. Sara' la fine dello sviluppo classico?
