# Cinquanta sfumature di Azure Compute!
##  13/04/2026 - Global Azure 2026 - Milano
### Abstract 
Hai un’app e vuoi portarla su Azure...
.... poi Azure ti guarda e dice: “Scegli.”

In questa sessione intro scopriremo insieme le principali piattaforme di esecuzione: Virtual Machines, App Service, Azure Functions, container runtime e AKS.

Un talk leggero ma concreto, pensato per sviluppatori e team che vogliono capire il compute Azure senza dolore.
