# Onboard Developers in 5 Minutes: Modern Dev Environments with Codespaces
##  02/07/2026 - DWX - Mannheim
### Abstract 
New teammate joins. You say: “Just click here and start coding.”
No setup scripts. No dependency mismatches. No tears.

This session explores how Codespaces + Dev Containers streamline onboarding and collaboration in teams.
We’ll build a shared configuration, test it, and discuss best practices for team-ready environments.
Your future teammates will thank you.
